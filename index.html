1. Ve a console.cloud.google.com → tu proyecto → APIs → Google Drive API → Credenciales → OAuth 2.0 → Aplicación web.
2. En "Orígenes de JavaScript autorizados" añade la URL donde tienes alojada la app (ej: https://tu-usuario.github.io).
3. No hace falta añadir URIs de redirección.
